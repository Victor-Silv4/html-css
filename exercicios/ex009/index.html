Tipos de listas em HTML5

Listas ordenadas

  1. item 1
  2. item 2

Listas não ordenadas

  • item a
  • item b

Listas mistas

Listas mistas misturam dois ou mais tipos de lista em uma só, por exemplo:

  • Jogos que já joguei
    1. GOW
    2. Pokemon
    3. Robloxkkkkk
  • Jogos que quero jogar
    1. Monster hunter
    2. Hogwarts Legacy
    3. Baldurs Gate

Listas de definição

Lista de dfinição recebm uma palavra e sua respectiva definição, como um dicionário

HTML
Linguagem de marcação usada para a definição da forma de um site
CSS
Linguagem de estilização usada para definir o estilo de um site
Javascript
Linguagem de programação usada para definir a interação do site
