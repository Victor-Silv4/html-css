Links

Links externos

Links externos, como o nome já evidencia, são links de outras páginas
Para criar um link clicável para outra página, basta usar a tag "a". Exemplo: Meu perfil no linkdin

Melhor canal de programação do youtube

Links internos

Links internos levam o usuário para arquivos que estão dentro da pasta do projeto, como outra página por exemplo

Links para download

Links para download são muito similares á links comuns, usamos a tag anchor, a unica difeença é nos parametros que você usa junto à tag

Por exemplo: baixar imagem do curso de referência

Em alguns navegadores, o arquivo pode abrir ao invés de começar o download, o ideal seria usar arquivos .zip
